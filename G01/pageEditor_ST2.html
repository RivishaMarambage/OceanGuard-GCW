📄 Pages Implemented

Splash Screen (splash.html)

The splash screen serves as the first point of interaction for users visiting the OceanGuard

  • Purpose:
    • Introduces the OceanGuard website and SDG 14 theme
    • Redirects users to home page automatically
  • Features:
    • Title, subtitle, and mission statement
    • Team member list
    • Loading animation (CSS spinner)
    • Countdown timer
    • Skip intro button
  • HTML:
    • Structured using headings, paragraphs, list, and button
    • Uses meta refresh for automatic redirect
  • CSS:
    • Full-screen ocean background
    • Gradient overlay for readability
    • Glassmorphism container
    • Animations (fade-in, bounce, spinner)
  • JavaScript:
    • Countdown timer updates every second
    • Skip button redirects to home page

Action Impact Simulator (ais.html)

An interactive page where users select ocean conservation actions and receive dynamic impact feedback:

  • Purpose: Allows users to select ocean-friendly actions and calculate their environmental impact.
  • Structure: Contains a hero section, a grid of six action cards, and a feedback section showing total score, impact level, and action buttons.
  • HTML: Each action card uses custom data attributes such as data-points and data-title.
  • CSS: Uses a grid layout, hover effects, borders, transitions, and colour-based feedback for different impact levels.
  • JavaScript: Includes functions to select and deselect cards, calculate the total score, update the feedback message, show the impact level, and reset the simulator.
  • Interactive Effect: The background image changes based on the final score to make the page more dynamic.

Content Page

  • Topic: Coral Reef Protection and Sustainability.
  • Structure: Includes a hero section, internal navigation menu, and four main sections on coral reefs, threats, effects on marine life, and student actions.
  • HTML: Uses section IDs so users can jump to different parts of the page through internal links.
  • CSS: Uses classes such as content-st_2 and Sub_content, with Flexbox to align text and images side by side.
  • Design: Combines text and images to make the page more engaging and easier to understand.
  • Purpose: Supports SDG 14 by explaining coral reef importance and showing realistic actions students can take.
